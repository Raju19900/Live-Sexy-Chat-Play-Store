# Project Summary
The Live Sexy Chat project is an innovative entertainment application designed for adults in India. It focuses on fostering meaningful interactions through live video chats, offering users a platform to connect and engage in a fun and safe environment.

# Project Module Description
The application consists of various functional modules that enhance user experience, including:
- **Live Video Streaming**: Enables real-time interaction between users.
- **User Profiles**: Allows users to create and customize their profiles.
- **Chat Features**: Provides text and video chat options.
- **Responsive Design**: Ensures compatibility across devices from mobile to desktop.

# File Description Inventory
- **index.html**: The main HTML file that serves as the entry point for the application.
- **styles/**: Directory containing all stylesheets for the application.
- **scripts/**: Directory containing JavaScript files that implement application logic.
- **assets/**: Directory for images and other media used in the application.

# Technology Stack
- **Frontend**: HTML, CSS, JavaScript
- **Build Tools**: pnpm, Lint for code quality checks

# Usage
To set up the project, follow these steps:
1. Install dependencies using `pnpm install`.
2. Run lint checks with `pnpm run lint`.
3. Build the project using `pnpm run build`.

After building, the application is fully optimized and ready for export or live deployment.
